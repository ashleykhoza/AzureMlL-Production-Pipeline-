# AzureMlL-Production-Pipeline-
Developed an end-to-end Azure ML pipeline, from workspace provisioning to model training and deployment, using both notebooks and scripts.
I started by setting up an Azure Machine Learning workspace using the Azure CLI—this gave me a centralized environment for managing all resources. Then, I created both a compute instance and a compute cluster so I could run experiments efficiently. After that, I cloned the lab repository, ran the setup script, and installed the Python SDK on the compute instance. I explored a notebook, trained a model, and converted it into a script so it could be reused. I then added parameterization to the script for flexibility, ran it in the terminal to verify accuracy, and finally executed it as a command job in Azure ML studio. Each step gave me a structured, repeatable pipeline that can easily scale for future ML projects.
